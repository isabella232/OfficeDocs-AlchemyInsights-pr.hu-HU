# <a name="using-custom-policy-in-intune"></a>Egyéni házirend használata az Intune-ban

Az Intune számos beépített beállítási lehetőséget tartalmaz Android, iOS, macOS és Windows eszközökhöz. Létrehozhat egy egyéni profilt is, majd az Intune használatával telepítheti ezeket az egyéni beállításokat az eszközeire. Az egyéni profilok olyan eszközbeállítások és funkciók hozzáadására vannak kialakítva, amelyek nem az Intune beépített funkciói.

[Egyéni beállítások Android-eszközökhöz az Intune-ban](https://docs.microsoft.com/intune/custom-settings-android)

[Egyéni beállítások androidos nagyvállalati eszközökhöz az Intune-ban](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Egyéni beállítások iOS-eszközökhöz az Intune-ban](https://docs.microsoft.com/intune/custom-settings-ios)

[Egyéni beállítások macOS-eszközökhöz az Intune-ban](https://docs.microsoft.com/intune/custom-settings-macos)

[Egyéni beállítások az Windows 10-eszközökhöz az Intune-ban](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Egyéni beállítások a Windows Holographic for Business Intune-ban](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
