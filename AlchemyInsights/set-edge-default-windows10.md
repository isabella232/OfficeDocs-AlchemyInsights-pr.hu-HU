# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>A Microsoft Edge beállítása alapértelmezett böngészőként Windows 10-es eszközön

Ezt a következőképpen teheti meg:

1. Válassza a **Start gombot,** majd írja be az **Alapértelmezett alkalmazások értéket.**

1. A találatok között válassza az Alapértelmezett **alkalmazások lehetőséget.**

1. A **Webböngésző alatt válassza** a Microsoft Edge **lehetőséget.**
